
# Fuel Agent Kit Examples

This directory contains example scripts demonstrating various functionalities of the Fuel Agent Kit.

## Available Examples

- **mira/addLiquidity.ts**: Demonstrates how to add liquidity to a pool on the Mira protocol.
- **mira/swap.ts**: Shows how to perform token swaps using the Mira protocol.
- **read/balance.ts**: Illustrates how to read and display the balance of an account.
- **swaylend/borrow.ts**: Demonstrates borrowing assets using the Swaylend protocol.
- **swaylend/supply.ts**: Shows how to supply assets to the Swaylend protocol.
- **transfers/transfers.ts**: Demonstrates how to transfer tokens between accounts.
